## 简介

### 多继承问题
> java中一个类不能直接继承两个类比如说这样：class A extends B,C
不能这样写，因为java不支持多继承，但是你可以像下面这样实现继承多个类
class A extends B
class C extends A
这样C就同时继承了B和A两个类。

> 在C++中，这个多继承就好解决的多了
```c++

    public:virtual void foo(){
        printf("A");
    }
}
```
