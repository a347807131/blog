### 分支改名

本地分支改名

```bash
#如果对于分支不是当前分支，可以使用下面代码：
git branch -m [old_branch_name] [new_branch_name]
#如果是当前，那么可以使用加上新名字
git branch -m [new_branch_name]
```

删除远程仓库分支

```bash
git push origin :[old_branch_name]
```

将改名后的分支push到远程

```bash
git push origin [new_branch_name]
```
