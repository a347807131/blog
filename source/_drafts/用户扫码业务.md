## 用户扫码
- 接入： `scan`
### 请求参数
```java
private long[] account;         //商户id(客户端可能传多个)

private String accessToken;     //用户token

private String JDBID;           //借贷宝id

private String memberID;          //借贷宝用户id

private String udid;

private String clientVersion;   //客户端版本
```
