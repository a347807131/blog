## 接入： `/api/merchant/gateway`

## 支付回调
- 接入： `pay-callback`
### 请求参数
```java
private String status;              //支付结果状态

private String returnCode;          //返回码

private String returnMessage;       //返回码信息提示

private String tradeNo;             //金佰仕系统返回的交易流水号

private String purchaserId;         //借贷宝ID

private String txMerchantId;        //真实交易的商户号

private String orderId;             //商户系统的订单号

private String orderTime;           //商户发起请求的日期

private String totalAmount;         //订单金额，单位为分

private String merchantAmount;      //商户优惠金额

private String merchantIncome;      //订单金额减去商户优惠金额的值

private String JDBAmount;           //借贷宝优惠金额

private String realPayAmount;       //用户实际支付金额

private String fee;                 //手续费，单位为分

private String payTime;             //支付时间，格式：YYYY-MM-DD HH:MM:SS

private String payMethod;           //支付方式 01-余额 10-银行卡

private String backParam;           //交易返回时原样给支付的信息

private String sign;                //签名

private String ts;                  //请求时的时间

private String appkey;              //应用标识

private String request_source;         //请求来源，1-H5，2-外部server，3-内网server，4-oauth
```

## 支付网关下单
- 接入： `generate-order`
### 请求参数
```java
private String third_order;     //商家订单号

private String mid;             //商家在商户平台的帐户ID

private String jdb_id;          //支付用户的UUID

private String pay_account;     //商家在支付平台的支付账号

private long amount;            //订单总金额，单位为分

private int use_favor;          //是否使用优惠，0-不使用 1-使用

private int consume_mode;       //消费场景 1-扫商户码 3-付款码 4-收银台

private String expire;          //订单过期时间，单位为分钟，默认为30分钟

private String favors;          //优惠信息，use_favor为1时有效，JSON格式

private String sign;            //签名

private String ts;              //请求时的时间

private String appkey;          //应用标识

private String request_source;  //请求来源，1-H5，2-外部server，3-内网server，4-oauth
```
