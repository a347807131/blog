## 接入： `/api/merchant/favor`

## 获取推荐优惠
- 接入： `get-recommend`
### 请求参数
```java
private long mid;               //商户帐户ID

private String jdb_id;          //用户借贷宝UUID

private long amount;            //订单总金额，单位为分

private String order;           //商户订单号，绑定订单

private String third_order;     //第三方订单号

private String sign;            //签名

private String ts;              //请求时的时间，时间戳

private String appkey;          //应用标识

private int request_source;     //请求来源，1-H5，2-外部server，3-内网server，4-oauth

private String consume_mode;       //消费场景 1-用户扫商户码 2-商户app扫用户码 3-商户扫码枪扫用户码 4-九宫格
```

## 获取卡卷列表
- 接入： `list`
### 请求参数
```java
private long mid;               //商户帐户ID

private String jdb_id;          //用户借贷宝UUID

private long amount;            //订单总金额，单位为分

private String page;            //页数，默认为1

private String limit;           //每页数目，默认为15

private int type;               //优惠类型 1-卡券 2-平台活动，默认为1

private String sign;            //签名

private long ts;                //请求时的时间，时间戳

private String appkey;          //应用标识	PAYGENT

private int request_source;     //请求来源，1-H5，2-外部server，3-内网server，4-oauth
```

## 获取优惠金额
- 接入： `off-amount`
### 请求参数
```java
private long mid; // 商家在商户平台的帐户ID

private String jdb_id; // 用户借贷宝UUID

private long amount; // 订单总金额，单位为分

private String favors; // 优惠信息，JSON格式

private String order; // 商户订单号，若传递，则绑定订单

private String sign; // 签名

private long ts; // 请求时的时间，时间戳

private String appkey; // 应用标识 PAYGENT

private int request_source; // 请求来源，1-H5，2-外部server，3-内网server，4-oauth

// 新增字段
private String third_order; // 第三方订单号

private String consume_mode; // 消费场景 1-用户扫商户码 2-商户app扫用户码 3-商户扫码枪扫用户码 4-九宫格
```
