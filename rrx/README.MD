# 服务器日志脚本

## 需求分析

>* 1.可以根据参数在服务器上进行日志查询
>* 2.可以查询以往的日志记录
>* 4.可以根据应用名对对应的应用服务器进行日志查询
>* 5.支持正则表达式
>* 6.支持将服务器列表存在放在配置文件中
>* 7.支持过滤掉指定应用查询或只查询指定服务器
